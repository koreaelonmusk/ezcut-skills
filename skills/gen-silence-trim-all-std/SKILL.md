# 무음 자동 트림 — 공통

말 사이 빈 공간을 모두 제거해 밀도를 높여.

## 프롬프트
말 사이 빈 공간을 모두 제거해 밀도를 높여.

## 언제 쓰나
말하는 영상에 적합. 패턴 인터럽트(3~5초마다 변화)는 도파민 리듬을 유지해 이탈을 막는다 — 정점에서 컷하면 인지 부하가 낮아 끝까지 본다.
