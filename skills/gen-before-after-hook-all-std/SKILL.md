# 비포-애프터 훅 — 공통

결과(애프터)를 먼저 1초 보여주고 과정으로 되돌아가.

## 프롬프트
결과(애프터)를 먼저 1초 보여주고 과정으로 되돌아가.

## 언제 쓰나
변화·튜토리얼에 적합. 조회수의 80%는 첫 3초에 갈린다 — 알고리즘은 초반 swipe-away(이탈률)를 노출 억제 신호로 읽으므로 hook rate를 최우선으로 설계한다.

# Before-After Hook — All (English)

Flash the result for 1s then rewind to the process.

## Prompt
Flash the result for 1s then rewind to the process.

## When to use
Great For transformations/tutorials. 80% of views are decided in the first 3s — the algorithm reads early swipe-away as a suppression signal, so hook rate is the #1 lever.
