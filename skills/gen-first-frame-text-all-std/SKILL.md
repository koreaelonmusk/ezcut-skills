# 첫 프레임 텍스트 — 공통

첫 프레임에 대표 한 줄을 크게 고정해 썸네일 겸 훅으로.

## 프롬프트
첫 프레임에 대표 한 줄을 크게 고정해 썸네일 겸 훅으로.

## 언제 쓰나
쇼츠·릴스에 적합. 조회수의 80%는 첫 3초에 갈린다 — 알고리즘은 초반 swipe-away(이탈률)를 노출 억제 신호로 읽으므로 hook rate를 최우선으로 설계한다.

# First-Frame Text — All (English)

Pin a strong line on the first frame as thumbnail+hook.

## Prompt
Pin a strong line on the first frame as thumbnail+hook.

## When to use
Great For Shorts/Reels. 80% of views are decided in the first 3s — the algorithm reads early swipe-away as a suppression signal, so hook rate is the #1 lever.
